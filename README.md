# RimTours Website Data Scrape and Clean

## Overview
Repository for scraping, cleaning, and processing RimTours website data including:
- Tour descriptions and metadata
- Pricing information
- Image URLs and filenames
- Calendar and availability data
- Business group classifications

## Purpose
This repository contains scripts and processes for extracting clean, structured data from the RimTours website for integration with Arctic reservation system and Outline documentation.

## Data Sources
- WordPress website export (ACF fields)
- Tour itinerary documents (Word files)
- Arctic system data
- Pricing calendars

## Output
- Cleaned tour data for Arctic integration
- Structured markdown files for documentation
- Calendar-ready scheduling information
- SEO-permalink preserved links
