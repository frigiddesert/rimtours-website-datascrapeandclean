Join us for this memorable sunrise tour, complete with a delicious continental-style breakfast, unparalleled scenic vistas and a fun descent (3500 feet elevation loss!) down into Moab. This tour is perfect for those without a lot of mountain biking experience. That said, all riders need to be comfortable using hand brakes as well as be fit enough to pedal up some rising terrain.
